HTML: Fundamentos del Marcado


¿Qué es HTML y para qué sirve?

HTML, o "HyperText Markup Language" (Lenguaje de Marcado de Hipertexto), es el lenguaje base de la web. Su función es definir la estructura y el contenido de una página web, permitiendo a los navegadores interpretar y mostrar el texto, las imágenes, los enlaces y otros elementos de una manera coherente.

Cada etiqueta tiene una función:

  • !DOCTYPE html: Indica al navegador que este documento es HTML5.
  • html y /html: Marcan el inicio y el fin del documento.
  • head y title: Contienen metadatos y el título de la página.
  • body: Aquí va el contenido que veremos en pantalla.

Etiquetas Comunes en HTML

Para trabajar con contenido básico, HTML ofrece una variedad de etiquetas que podemos usar. Aquí están algunas de las más comunes:

  1. Encabezados (h1 hasta h6): Ayudan a jerarquizar la información.
  2. Párrafos (p): Para el texto de los párrafos.
  3. Enlaces (a href="url"): Permiten vincular a otras páginas o recursos.
  4. Imágenes (img src="url">): Insertan imágenes en la página.
